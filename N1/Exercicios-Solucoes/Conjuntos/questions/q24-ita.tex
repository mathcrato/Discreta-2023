\item
(ITA 2002) Sejam $A$ um conjunto com $8$ elementos e $B$ um conjunto tal que $A \cup B$ contenha $12$ elementos. Então, o número
de elementos de $P(B - A) \cup P(\varnothing)$ é igual a
\begin{multicols}{5}
\begin{enumerate}
\item $8$
\item $16$
\item $20$
\item $17$%letra B
\item $9$
\end{enumerate}
\end{multicols}
