\item  Se $f$  uma função de $\mathbb{R}$ em $\mathbb{R}$ definida por
  $f(x)=(x-3)/(x^2+3)$, então a expressão $\frac{f(x)-f(1)}{(x-1)}$,
  para $x\leq 1$,  equivalente a
  \begin{multicols}{2}
	\begin{enumerate}
	  \item  $(x + 3)/2(x^2 + 3)$
	  \item $(x - 3)/2(x^2 + 3)$
	  \item $ (x + 1)/2(x^2 + 3)$
	  \item $ (x - 1)/2(x^2 + 3)$
	  \item $-1/x$
	\end{enumerate}
  \end{multicols}
